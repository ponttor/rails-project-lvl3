# Bulletin Board

*Within educational program 'Ruby on Rails developer' by [Hexlet](https://ru.hexlet.io/)*

## System dependencies

- Ruby 3.0.1
- Node.js
- Yarn
- SQLite3
- Heroku CLI

## Installation

```ruby
make setup
make test # run tests
make start # run server

make seed # load fixtures
```
